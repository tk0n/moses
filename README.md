# moses ~ Automated Snapshots for [ArkEcosystem/ark-node](https://github.com/ArkEcosystem/ark-node)

## Installation

```bash
git clone https://github.com/faustbrian/moses
cd ~/moses
bash moses.sh install
bash moses.sh start
```
